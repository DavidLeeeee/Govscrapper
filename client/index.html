정부지원사업공고 페이지 사용법입니다.

01

공고조회와 지역공고

공고조회는 주요 정부지원사업을, 지역공고는 지역 중심 공고를 따로 보여줍니다. 지역공고는 정보 수집용으로 분리되어 일반 마감공고 페이지에는 섞이지 않습니다.

02

키워드 검색

검색창은 공백 기준으로 여러 단어를 검색합니다. 빠른검색어는 브라우저에 저장되며, 공고조회와 지역공고에서 같은 목록을 공유합니다.

03

공유 북마크

카드의 별 버튼을 누르면 모든 사용자가 보는 공유 북마크에 저장됩니다. 마감일이 없지만 계속 보관하고 싶은 공고는 북마크해두는 것을 권장합니다.

04

상세 모달

공고 카드를 누르면 요약, 예산액, 핵심 내용, 원문 링크를 확인합니다. 공유 버튼은 Google Chat으로 해당 공고를 전달하고, 마감시키기는 목록에서 직접 제거할 때 사용합니다.

05

AI 심층분석

첨부파일을 기반으로 사업 목표, 배경, 필요성, 개발 내용, 요구사항, 구현 포인트, 추천 액션을 정리합니다. 이미 분석된 공고는 저장된 결과를 다시 보여주어 불필요한 API 호출을 줄입니다.

06

마감일 없는 공고

마감일이 없는 공고는 기본적으로 30일 뒤 정리됩니다. 이 기간은 서버 환경값 NO_DEADLINE_EXPIRE_DAYS로 변경할 수 있습니다.
