도움말

정부지원사업공고 페이지 사용법입니다.

01

공고조회

수집된 공고를 카드 형태로 확인합니다. 상단 필터를 누르면 사이트별 공고만 볼 수 있습니다.

02

키워드 검색

검색창에 입력한 키워드는 공고 제목과 비교됩니다. 하단 shortcut 키워드를 누르면 해당 키워드로 바로 검색됩니다.

03

공유 북마크

카드의 별 버튼을 누르면 공고가 공유 북마크로 저장됩니다. 공고가 정리된 뒤에도 북마크 탭에서는 모든 사용자가 같은 북마크 목록을 확인합니다.

04

마감일 없는 공고

마감일이 없는 공고는 기본적으로 60일 뒤 정리됩니다. 이 기간은 서버 환경값 NO_DEADLINE_EXPIRE_DAYS로 변경합니다.
